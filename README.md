# Tony Wilberg — Portfolio

Public portfolio site for **Tony Wilberg**, AI Platform Engineer.

A self-hosted, multi-agent AI platform (**DreamBox**) plus two autonomous systems built on
it (**Dereth** — ML / computer vision / navigation; **TradeBot** — disciplined, paper-first
autonomous trading). Twenty years of IT operations behind it.

## Stack

Single self-contained static HTML page — no build step, no dependencies.
Deployable to any static host.

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deploy

- **Cloudflare Pages / Netlify:** connect this repo; no build command; output directory = repo root.
- **GitHub Pages:** enable Pages on the default branch, root folder.
